# AI뉴스 요약 대시보드

실시간 뉴스 RSS 수집 + GPT-4o-mini AI 요약 + 챗봇이 통합된 한국어 뉴스 대시보드입니다.

**라이브 데모**: https://inukhwang1020.github.io/ai-news-dashboard/

---

## 주요 기능

### 뉴스 탭 (경제 / 테크 / 리스크 관리)
- 카테고리별 3개 RSS 피드를 병렬 fetch, 최대 9건 표시
- 제목 앞 20자 기준 중복 제거 후 최신순 정렬
- 카드 UI: 번호 배지 · 출처 · 시간 · 굵은 제목 · AI 요약 박스 · 원본 보기/북마크/토론 버튼

### AI 뉴스 요약
- OpenAI GPT-4o-mini로 카테고리 전체 기사를 한 번에 요약
- 기사당 3줄 불릿 요약, 카드 내 AI 요약 박스에 표시
- 헤더에 AI 상태 표시 (미설정=노란점, 활성=초록점)

### 챗봇
- 현재 로드된 뉴스 헤드라인 + AI 요약을 컨텍스트로 포함
- 대화 히스토리 최대 12턴 유지
- 화면 우측 상단 고정 패널 (탭 전환해도 닫히지 않음)

### 아카이브
- 뉴스 fetch 시 Supabase에 자동 저장 (batch_id 단위)
- 사이드바 3단계 트리: 카테고리 → 날짜 → 수집 시각
- 배치 클릭 시 해당 시점 기사 목록 복원

### 토론 게시판
- 닉네임 + 내용 작성, 좋아요, 수정(inline), 삭제
- 뉴스 카드 "토론" 버튼으로 기사 연결 가능

### 북마크
- 기사별 북마크 토글 (localStorage 저장)
- 북마크 모아보기 뷰

## 기술 스택

| 항목 | 내용 |
|---|---|
| 프론트엔드 | HTML · CSS · JavaScript (단일 파일) |
| UI 프레임워크 | Tailwind CSS CDN |
| AI | OpenAI GPT-4o-mini (요약 · 챗봇) |
| 데이터베이스 | Supabase (news_archive · discussions 테이블) |
| RSS 프록시 | corsproxy.io → allorigins.win → codetabs.com (폴백) |
| 배포 | GitHub Pages (main 브랜치) |

---

## 사용 방법

1. 라이브 데모 접속
2. 첫 화면에서 OpenAI API 키 입력 (또는 "나중에"로 건너뜀)
3. 경제 / 테크 / 리스크 탭에서 실시간 뉴스 확인
4. AI 요약이 자동으로 각 카드에 표시됨
5. 헤더 "챗봇" 버튼으로 뉴스 기반 Q&A 가능

> API 키는 브라우저 localStorage에만 저장되며 외부로 전송되지 않습니다.

## 로컬 실행

별도 빌드 없이 `index.html`을 브라우저에서 열거나 라이브 서버로 실행합니다.

```bash
# 예: VS Code Live Server 또는
npx serve .
```

> RSS fetch는 CORS 프록시를 사용하므로 로컬에서 일부 피드가 차단될 수 있습니다. 정상 동작 확인은 GitHub Pages 배포 주소를 권장합니다.
